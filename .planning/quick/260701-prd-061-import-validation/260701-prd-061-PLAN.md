# PRD-061 Import Validation

## Goal

Add backend validation for weekly menu uploads so template, customer, week, dish mapping, and duplicate-row issues are returned as a structured validation DTO with row/column/cell details where available.

## Scope

- Extend weekly menu import result DTO with validation summary and issue rows.
- Return validation DTO for unknown customer and parser-level critical failures during preview.
- Mark week-start mismatch and duplicate import slots as critical errors.
- Mark unknown dishes as row/cell warnings before commit.
- Block commit when validation has critical errors so no partial data is written.
- Surface validation issues in the import wizard.
